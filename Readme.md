# Rap接口转YApi

## 前言

本项目主要目的是提供Rap接口转YApi接口的解决方案，支持对Rap、Rap2版本的接口转成Swagger格式json，YApi json格式，从而进行接口迁移。

功能：

* Rap接口转 Yapi json
* Rap2接口转 Swagger json
* Rap2接口转 Yapi json
